SEC-001 — Azure SQL has public endpoint enabled
sql-contoso-prod · rg-contoso-data-prod
🛡️ Security ›

What was found

Azure SQL Server sql-contoso-prod has publicNetworkAccess: Enabled with no Private Endpoint configured. The database is accessible from the public internet, relying only on firewall rules and authentication for protection.

Why it matters

Public endpoints expose databases to brute-force attacks, credential stuffing, and potential data exfiltration. Even with strong authentication, the attack surface is unnecessarily large.

Recommendation

Deploy a Private Endpoint for the SQL Server in the snet-data subnet, create the required Private DNS Zone, and disable public network access.

What was found
